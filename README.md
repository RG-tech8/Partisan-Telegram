## Partisan-Telegram messenger for Android


![](https://github.com/RG-tech8/Partisan-Telegram/blob/master/wiki_images/readme.webp)

RGram — специальная версия Telegram для пользователей в России и других несвободных странах, которым нужен безопасный режим на случай давления.
У RGram несколько паролей: основной и ложные. Ложные запускают заранее заданные действия, например:
- отправка пользовательского SOS‑сообщения близким и доверенному контакту через Telegram
- удаление чатов и каналов, которые могут навредить пользователю
- выход из аккаунта на этом устройстве
- завершение всех остальных сессий, кроме текущей
- и другие

Использование RGram само по себе может стать риском, поэтому приложение выглядит и ощущается максимально похоже на обычный Telegram.

### Шифрование (RGCRYPT)

RGram использует модуль RGCRYPT для шифрования сообщений поверх обычных чатов. Это гибридная схема:
- содержимое сообщения шифруется AEAD (AES‑256‑GCM);
- для каждого сообщения создаётся случайный ключ, который доставляется получателям через HPKE (X25519 + HKDF‑SHA256 + AES‑256‑GCM);
- подпись Ed25519 защищает KeyCard (карточку с публичными ключами), чтобы обнаруживать подмену.

Каждое RGCRYPT‑сообщение также подписывается ключом отправителя. Проверка подписи выполняется, если в базе есть публичный ключ отправителя; без него подпись помечается как неизвестная. Доверие к ключу устанавливается вручную после сверки отпечатка/safety number.

Важно: RGCRYPT защищает содержимое, но не метаданные Telegram‑диалога (кто, кому и когда написал). Шифротекст всё равно проходит через обычный чат.
